How to Play...

The aim of the game is to guess which room each picture was taken correctly.
Every round you will be presented with a photo and you must select on the given
floor plan map of the building where the picture was taken. Each correct answer
will score you many points but BEWARE, wrong answers will take away 5 points.
At the end of the game don't forget to add yourself to the leaderboards.
